# 专题 02：函数指针与回调

HAL 库所有外设回调、STM32 中断向量表，全是函数指针。

## 题目获取

`FuncPtr.h` 就是题目说明书。打开它，按题号顺序做题。

## 题目总览

10 道题，分四幕：

- 幕一：函数指针基础（题1-3）— 声明调用、typedef、返回函数指针
- 幕二：进阶应用（题4-6）— 命令分发表、找Bug、复杂声明阅读
- 幕三：const 与 void*（题7-9）— const指针、泛型上下文、硬件地址访问
- 幕四：综合（题10）— HAL 外设回调注册模拟
